# Stresless aka BearSmash

Throw a rubber bear with ESP8266 towards moving target. ESP8266 measures acceleration, while camera detects target colour and bear colours.

***
# Development

Run flask server

`FLASK_APP=app.py flask run --host 0.0.0.0`

Browser

`0.0.0.0:5000/hit`
